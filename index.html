Calling API using Express

Exchange Rate

Latest Exchange Rate

http://localhost:5000/latest

Exchange Rate on Date(YYYY-MM-DD)

http://localhost:5000/:date

Exchange Rate at Base currency(e.g. AUD)

http://localhost:5000/base/:base

Exchange Rate of selected currency against base currency

http://localhost:5000/latest/symbols/:symbols

Exchange Rate between dates

http://localhost:5000/history/startdate/:sdate/enddate/:edate

Exchange Rate of selected currency against base currency between dates

http://localhost:5000/history/startdate/:sdate/enddate/:edate/symbols/:symbols

Exchange Rate of base currency between dates

http://localhost:5000/history/startdate/:sdate/enddate/:edate/base/:base
